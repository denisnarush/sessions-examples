### Align block to the right side

HTML:
```
<div class="container">
  <div class="block">
</div>
```
CSS:
```
.container {
  border: 3px dashed rgba(0, 0, 0, .2);
}

.block {
  background-color: #96C6FF;
  border: 3px solid #C2E2FA;
  height: 60px;
  width: 31.5%;
}
```
PRECONDITIONS:
- container width: any
- container hegith: any
- block width: any
- block height: any
- don't use extra html
- don't use extra css selectors
